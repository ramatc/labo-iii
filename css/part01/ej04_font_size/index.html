Tamaño de las fonts

Los textos marcados con "H1" tendrán dimensión doble a los párrafos comunes. Corresponderían a titulos.

En el ejercicio se definirá un tamaño de letra default para el body y luego todo será expresado en unidades em (unidad de referencia).

En este ejercicio se trabajará con control de medio @media() para que el tamaño de letra sea modificado según el tamaño del viewport

Dispositivos moviles de alta resolución tienen una definición mayor a: 300ppi (Pixels/pulgada);

Dispositivos moviles de baja resolución tienen una definición menor en el orden de los 100ppi (pixeles/pulgada)

Dispositivos de pantalla grande tienen una definición en el orden de los 60 (pixeles/pulgada)

En un movil de alta resolución estaría viendo las letras de 3 a 5 veces mas chicas que en una pantalla grande si su tamaño es fijo en pixeles

Por este motivo es que si no uso el control de medio, será más conveniente definir el tamaño de letra considerando el peor de los casos (los dispositivos de alta resolucion)

En este ejercicio hago crecer la letra conforme el dispositivo es mas pequeño, siempre suponiendo que estos tienen mayor resolución

12px (1em) para dispositivos mayores a 1080px. No olvidar definir el tamaño default en 12px

24px (2em) para dispositivos mayores a entre 800px y 1080px

36px (3em) para dispositivos menores a 800px
