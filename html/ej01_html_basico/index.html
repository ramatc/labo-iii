Preguntas:

  1. Dar un par de ejemplos de uso de etiquetas meta.

    • Los meta tags con el atributo “http-equiv” pueden contener información equivalente a las indicaciones en el encabezado HTTP. <"meta http-equiv="cache-control" content="no-cache" />
    • Los meta tags con el atributo "description" pueden describir brevemente el contenido de la página web. <"meta name="description" content="..." />

  2. Nombrar al menos 2 atributos aplicables a la etiqueta HTML.

    • "lang": permite especificar el idioma principal utilizado en el contenido de una página web.
    • "id": permite identificar de manera única un elemento HTML.
    • "class": permite asignar una o varias clases a un elemento.

  3. ¿Como se relaciona un header http con la etiqueta head de HTML?

    Los header http se relacion con las etiquetas meta, ya que estas identifican la metadata de una página web, y en este caso, el atributo "http-equiv" con valor "Content-type" y el atributo "content" con valor "utf-8" le indican al navegador que se trata de un encabezado HTTP llamado "Content-type" y que su contenido o valor es el código de lenguaje español "utf-8". La persona o proceso que codifica el HTML probablemente no pueda generar un encabezado HTTP y lo guarda de esta manera utilizando las etiquetas meta. El uso de etiquetas meta debe ser cuidadosamente considerado en función del público objetivo y el contenido de la página web.

  4. ¿Que es UTF8?

    UTF-8 (UCS Transformation Format 8) es la Codificación de caracteres más común en la red. El número de bytes que representan un carácter pueden ser desde uno hasta cuatro. UTF-8 es retrocompatible con ASCII y puede representar cualquier carácter Unicode estandar.

  5. ¿Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?

    El atributo style es un atributo que se utiliza en las etiquetas HTML para incrustar código CSS directamente en la propia etiqueta. Este tipo de CSS se denomina Estilos CSS en línea, y puede ser aplicado a cualquier etiqueta HTML.

  6. ¿Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?

    Para indicar que un enlace debe abrirse en una nueva pestaña, se utiliza el atributo "target" con el valor "_blank" a la etiqueta "a". De la siguiente manera: "a href="https://..." target="_blank".

  7. Explicar cuando usarían etiquetas semánticas y con que propósito.

    Las etiquetas semánticas de HTML5 se utilizan para proporcionar información adicional sobre el contenido de una página web, y su propósito es ayudar a los motores de búsqueda a comprender mejor el contenido de la página. Estas etiquetas marcan los límites de distintos tipos de contenido en una página web, y se les llama semánticas porque no tienen nada que ver con la estructura sintáctica ni con el estilo del documento, sino que se enfocan pura y exclusivamente en el contenido. Se recomienda utilizar etiquetas semánticas siempre que sea posible, ya que esto puede mejorar la accesibilidad y la legibilidad de la página. En conjunto, las etiquetas semánticas otorgan una estructura semántica a nuestro documento, lo que facilita su comprensión tanto para los motores de búsqueda como para los usuarios que lo visitan. Por ejemplo, para contener la lista de navegación o enlaces de nuestro documento, hay que utilizar la etiqueta única "nav".

  8. ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.

    Los atributos de estilo básicos que aplicarían a una tabla son:

    • border: para establecer el grosor del borde de la tabla.
    • border-collapse: se utiliza para fusionar los bordes.
    • width: define el ancho de la tabla en pixels o porcentaje.
    • height: define la altura de la tabla en pixels o porcentaje.
    • background-color: para establecer el color de fondo de la tabla.

  9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

    El body de un documento HTML es la sección que contiene el contenido principal del documento, como texto, imágenes, enlaces, etc. El body del protocolo HTTP se refiere a la sección del mensaje HTTP que contiene los datos de la respuesta, enviados por el cliente al servidor o por el servidor al cliente. El cuerpo de un mensaje HTTP puede contener diferentes tipos de datos, y su formato puede variar según el tipo de datos que se estén enviando.

  10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?

    Los atributos que marcan la agrupación de celdas de una tabla son colspan y rowspan. colspan se utiliza para combinar varias celdas en una sola columna, mientras que rowspan se utiliza para combinar varias celdas en una sola fila. Estos atributos se aplican a las etiquetas "td" y "th".
